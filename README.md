# Responsive-issues-Media-queries-and-variable
# How to Run

1. Download all project files.
2. Open index.html in a browser.
3. Resize the browser window to test responsiveness.

# Technologies Used

- HTML5
- CSS3
- Flexbox
- Media Queries

# Responsive Features

- Flexbox navbar
- Tablet breakpoint (768px)
- Mobile breakpoint (425px)
- Navigation hidden on mobile
- Hero section changes from row to column layout
